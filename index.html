Margins

Com o Structure CSS você não vai precisar mais escrever um código no css para colocar alguma margem em seu projeto, basta colocar uma class com o seguinte código mg-t-*, mg-b-*, mg-l-* e mg-r-*.

Os valores vão e 5 pixels a 90 pixels.


Classes

mg-t-* = margin-top: *;

mg-b-* = margin-bottom: *;

mg-l-* = margin-left: *;

mg-r-* = margin-right: *;
